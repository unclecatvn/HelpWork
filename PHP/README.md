# Các câu lệnh cơ bản của laravel

- Chạy composer

```php
composer install
```

- Tạo file .env rồi tạo key:

```php
cp .env.example .env
php artisan key:generate
```

- Chạy nodejs:

```php
npm install
npm run watch
```

- Chạy server

```php
php artisan serve
```

- Cài đặt laravel Mix và cài đặt xử lí môi trường

```php
npm install laravel-mix --save-dev
npm install cross-env --save-dev
```
